📌 Method 2: DevTools (Always Works)

  1. Open the video page in Chrome
  2. Press F12 → Network tab
  3. Start playing the video
  4. In the filter box, type m3u8
  5. Right-click the m3u8 URL → Copy → Copy link address
  6. Paste into Video Downloader in 🔗 URL mode

🎯 Pro tip: You can also filter for .ts or .mp4 to find video segments.

📌 Method 3: Local Capture Server

Run this command to start a local capture server:

python3 backend/downloader.py helper-server --port 9876

Then set your browser's proxy to 127.0.0.1:9876 and browse — all m3u8/mp4 URLs will be captured automatically.
